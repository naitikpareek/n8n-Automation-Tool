## 📌 WhatsApp API Integration – n8n

### 🔹 Overview

This project demonstrates a WhatsApp automation **workflow** built using **n8n**. The workflow receives incoming WhatsApp messages, structures the data, stores it in Google Sheets, and prepares automated responses using the WhatsApp API.

---

## 🔄 Workflow

### **Workflow Flow**

```
WhatsApp Trigger
      ↓
Edit Fields (Set)
      ↓
Google Sheets (Append Row)
      ↓
WhatsApp Send Message
```

---

### **WhatsApp Trigger**

* Listens for incoming WhatsApp messages.
* Captures message payload such as sender details and message content.
* Workflow execution is currently paused due to incomplete WhatsApp Meta API configuration.

---

### **Edit Fields (Set)**

* Structures incoming data into a clean and usable format.
* Extracts required fields:

  * Sender name
  * Phone number
  * Message text
* Ensures consistent data for downstream nodes.

---

### **Google Sheets – Append Row**

* Appends structured message data into a Google Sheet.
* Maps workflow fields directly to sheet columns.
* Enables real-time logging of WhatsApp messages.

---

### **WhatsApp Send Message**

* Prepares automated replies to WhatsApp users.
* Configured to send messages dynamically based on incoming data.
* Message delivery pending due to missing Meta API credentials:

  * Access Token
  * Phone Number ID
  * WhatsApp Business Account setup
